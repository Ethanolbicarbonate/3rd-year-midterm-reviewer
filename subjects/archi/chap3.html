Chapter 3: The Digital Logic Level

Gates and Boolean Algebra

The digital logic level is the computer's true hardware, built from simple electronic elements. A digital circuit is one where only two logical values are present (e.g., 0 and 1). These circuits are constructed from primitive elements called gates.

Gates

A gate is a tiny electronic device that can compute various functions of two-valued signals. They are the fundamental hardware basis on which all digital computers are built. All modern digital logic rests on the fact that a transistor can act as a very fast binary switch.

NOT Gate (Inverter)

Outputs the logical inverse of its single input.

NAND Gate

Outputs 0 if and only if all inputs are 1.

NOR Gate

Outputs 1 if and only if all inputs are 0.

AND Gate

Outputs 1 if and only if all inputs are 1.

OR Gate

Outputs 1 if at least one input is 1.

üí° Completeness: NAND and NOR gates are considered "complete" because any Boolean function can be constructed using only one of these gate types.

Boolean Algebra

Boolean Algebra is a mathematical technique used to analyze and describe the behavior of digital circuits. In this algebra, variables and functions can only take on the values 0 and 1.

  • Boolean Function: A function with one or more binary inputs that produces a binary output.
  • Truth Table: A table that completely describes a Boolean function by listing the output for every possible combination of inputs.

Basic Digital Logic Circuits

Integrated Circuits (ICs)

Gates are manufactured in units called Integrated Circuits (ICs) or chips. An IC is a rectangular piece of silicon containing a number of gates. Common packages for ICs include DIP (Dual Inline Package), PGA (Pin Grid Array), and LGA (Land Grid Array).

Combinational Circuits

A combinational circuit is a circuit with multiple inputs and outputs where the outputs are uniquely determined by the *current* input values. It has no memory of past states.

Multiplexer (MUX)

A circuit with 2‚Åø data inputs, n control inputs, and one data output. It selects one of the data inputs to be passed to the output based on the value of the control lines.

Demultiplexer (DEMUX)

The inverse of a multiplexer. It routes its single input signal to one of 2‚Åø outputs based on the value of its n control lines.

Decoder

A circuit that takes an n-bit number as input and selects exactly one of its 2‚Åø output lines.

Comparator

A circuit that compares two input words and outputs a signal indicating whether they are equal.

Arithmetic Circuits

Shifter

A circuit that takes an n-bit input and produces an output that is the input shifted by one or more bit positions to the left or right.

Half Adder

A circuit for adding two single bits. It produces a sum bit and a carry bit.

Full Adder

A circuit for adding three single bits (two input bits plus a carry-in bit). It forms the building block for multi-bit adders.

Arithmetic Logic Unit (ALU)

A circuit that can perform various arithmetic (e.g., addition) and logical (e.g., AND, OR) operations on n-bit words.

Memory

Memory is an essential component of every computer, used for storing both instructions and data. Unlike combinational circuits, memory circuits are sequential‚Äîtheir output depends on past inputs as well as current ones.

Latches and Flip-Flops

Latch

A circuit that can "remember" previous input values. An SR latch has Set (S) and Reset (R) inputs. A D latch samples its D (Data) input when a clock signal is active. Latches are level-triggered, meaning they are sensitive to the state of the clock (e.g., high or low).

Flip-Flop

A circuit that samples and stores an input value at a specific instant in time, typically on the rising (0 to 1) or falling (1 to 0) edge of a clock signal. Flip-flops are edge-triggered.

Registers

A register is a group of flip-flops combined to hold data types larger than 1 bit (e.g., an 8-bit register is made of 8 flip-flops).

Memory Organization and Chips

  • RAM (Random Access Memory): Memory that can be both read from and written to.
    • SRAM (Static RAM): Fast, uses flip-flops, retains data as long as power is applied. Used for cache memory.
    • DRAM (Dynamic RAM): Slower, uses capacitors, must be refreshed every few milliseconds. Used for main memory. Variants include SDRAM and DDR SDRAM.
  • ROM (Read-Only Memory): Nonvolatile memory whose contents are set during manufacturing and cannot be changed. Variants include PROM, EPROM, EEPROM, and Flash Memory.

CPU Chips and Buses

Modern CPUs are contained on a single chip and communicate with memory and I/O devices through a set of pins connected to a bus.

CPU Pin Categories

  • Address Pins: Used to specify a memory location. A CPU with 'm' address pins can address up to 2m memory locations.
  • Data Pins: Used to transfer data to and from memory.
  • Control Pins: Regulate the flow and timing of data and handle functions like bus control, interrupts, and bus arbitration.

Computer Buses

A bus is a common electrical pathway between multiple devices. Devices on a bus are either masters (which can initiate transfers) or slaves (which respond to requests).

Synchronous Bus

Has a master clock line. All bus activities are synchronized to this clock and take an integral number of clock cycles.

Asynchronous Bus

Does not have a master clock. It uses a handshaking protocol (e.g., MSYN/SSYN signals) to coordinate transfers between master and slave devices.

Bus Arbitration

A mechanism needed to decide which device gets to be the bus master when multiple devices request the bus simultaneously. A common method is daisy chaining, which grants access based on a device's physical proximity to the arbiter.

üìù Quick Review & Memorization

Key Terms & Acronyms

  1. Gate: The basic electronic element of a digital circuit (e.g., AND, OR, NOT).
  2. Boolean Algebra: The mathematical system used to analyze digital circuits.
  3. Truth Table: A table defining a logic function's output for all possible inputs.
  4. Combinational Circuit: A circuit whose output depends only on its current inputs (e.g., adder, multiplexer).
  5. Sequential Circuit: A circuit whose output depends on both current and past inputs (i.e., it has memory).
  6. ALU (Arithmetic Logic Unit): A circuit that performs arithmetic and logical operations.
  7. Latch: A 1-bit memory element that is level-triggered.
  8. Flip-Flop: A 1-bit memory element that is edge-triggered.
  9. Register: A collection of flip-flops used to store a multi-bit word.
  10. SRAM (Static RAM): Fast, volatile memory made of flip-flops; used for caches.
  11. DRAM (Dynamic RAM): Slower, denser, volatile memory made of capacitors; used for main memory.
  12. ROM (Read-Only Memory): Nonvolatile memory that cannot be written to after manufacturing.
  13. Bus: A shared communication pathway connecting multiple devices.
  14. Synchronous Bus: A bus whose operations are tied to a central clock.
  15. Asynchronous Bus: A bus that uses handshaking signals instead of a central clock.

Summary Points

  1. The Digital Logic Level is the hardware foundation of a computer, built from simple logic gates.
  2. Boolean Algebra is the tool used to describe and simplify the logic of digital circuits.
  3. Combinational circuits like multiplexers and decoders compute functions based on current inputs, while sequential circuits like latches and flip-flops store information, forming the basis of memory.
  4. An ALU is a core combinational circuit that performs arithmetic and logic operations.
  5. Memory is built from basic 1-bit storage elements (latches/flip-flops) organized into addressable words. Main memory is typically slower DRAM, while fast cache memory is SRAM.
  6. CPUs, memory, and I/O devices communicate over a bus, which can be either synchronous (clock-based) or asynchronous (handshake-based).
  7. Bus arbitration is required to manage access when multiple devices (masters) want to use the bus at the same time.
